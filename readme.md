# Linguagem HTML

## Tags básicas

A estrutura básica de um documento escrito em **HTML** contém as **tags** delimitadoras do conteúdo que será tratado pela linguagem **HTML**, uma secção de cabeçalho (**head**) e uma secção de corpo (**body**):

- **\<html\> e \</html\>:** Delimita o conteúdo do documento que será tratado como HTML.

- **\<head\> e \</head\>:** Secção de cabeçalho do documento responsável pelas informações de configuração do documento.

- **\<body\> e \</body>:** Secção do documento responsável pela parte visual do documento.

- **Exemplo:** Estrutura básica de um documento escrito em HTML.

    ```html
    <html>
        <head>
        </head>
        <body>
        </body>
    </html>
    ```

## Tags para adição e formatação de textos

- **\<p\> e \</p\>:** Inclui um parágrafo de texto.
- **Tags para separação e estilos**
    - **\<hr\>:** Inclui uma linha horizontal que separa tematicamente o conteúdo. É uma tag vazia (autcontida).
    - **\<br/\>:** Inclui uma quebra linha, forçando o texto seguinte a aparecer na linha de baixo.
    - **CSS (Cascading Style Sheets):** A forma mais flexível e recomendada para estilizar. Você pode aplicar estilos aos elementos `<div>`, `<span>` e outros, e definir o visual sem alterar a estrutura do HTML.
- **Cabeçalhos de uma página:** Níveis de importância.
    - **\<h1\> e \</h1\>:** Deve ser utilizado apenas uma vez por página. Identifica o assunto principal do conteúdo.
    - **\<h2\> e \</h2\>:** Usado para subtítulos que dividem o conteúdo em secções principais. Pode haver múltiplos **\<h2\>** em uma página.
    - **\<h3\> e \</h3\>:** Usado para subseções dentro de uma secção **\<h2\>**.
    - **\<h4\> e \</h4\>:** Usado para subdivisões ainda menores dentro de uma secção **\<h3\>**.
    - **\<h5\> e \</h5\>:** Usado para subdivisões dentro de uma secção **\<h4\>**.
    - **\<h6\> e \</h6\>:** Usado para o nivel mais baixo de cabeçalho, subdividindo secções **\<h5\>**.
- **\<strong\> e \</strong\>:** Formata o texto para **negrito** ou **bold**.
- **\<i\> e \</i\>:** Formata o texto para *itálico* ou *italic*.
- **\<u\> e \</u\>:** Formata o texto para **sublinhado** ou **underline**
- **\<sup\> e \</sup\>:** Formata o texto para **sobreescrito** ou **superscript**.
- **\<sub\> e \</sup\>:** Formata o texto para **subscrito** ou **subscript**.
- **Exemplo:** Texto dentro da secção **\<body\>** com as formatações acima.

    ```html
    <body>
        <strong>Meu primeiro <i>HTML!</i></strong>
        <strong><u>tecnologia<sup>*</sup>
    </body>
    ```

## Emmet abbreviation (Abreviações Emmet)

- `HTML:5`: Digite este comando seguido da tecla `TAB` para inserir uma estrutura padrão:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
```
